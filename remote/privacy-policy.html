Steelhead Automator — Política de Privacidad

Última actualización: 30 de marzo de 2026

Resumen

Steelhead Automator es una extensión de Chrome para uso interno del equipo de Ecoplating. No recopila, transmite ni almacena datos personales de los usuarios.

Datos que procesa la extensión

  • Archivos CSV: Los archivos que el usuario selecciona se leen localmente en el navegador y se envían directamente a la API de Steelhead ERP (app.gosteelhead.com). Ningún dato del archivo se transmite a terceros ni se almacena fuera del navegador.
  • Configuración remota: La extensión descarga archivos de configuración y scripts desde GitHub Pages (oviazcan.github.io). Estas descargas no contienen ni transmiten datos del usuario.
  • Cache local: La extensión almacena en chrome.storage.local únicamente la versión de configuración para verificar actualizaciones. No se almacenan datos personales ni contenido de archivos.

Datos que NO recopila

  • Información personal identificable
  • Historial de navegación
  • Cookies o credenciales
  • Datos de uso o analítica
  • Información de otros sitios web

Permisos de la extensión

  • activeTab: Para interactuar con la pestaña activa de Steelhead ERP.
  • storage: Para almacenar la versión de configuración localmente.
  • scripting: Para ejecutar los scripts de automatización en la página de Steelhead ERP.
  • host_permissions (app.gosteelhead.com): Para comunicarse con la API de Steelhead ERP.

Terceros

La extensión no comparte datos con terceros. Los únicos servicios externos que utiliza son:

  • GitHub Pages: Para descargar archivos de configuración y scripts (solo lectura, sin envío de datos del usuario).
  • Steelhead ERP: La API del sistema ERP al que el usuario ya tiene acceso autenticado.
